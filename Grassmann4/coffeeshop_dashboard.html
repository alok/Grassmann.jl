Ganja.js Coffeeshop Examples in Lean

This page renders the geometric algebra examples ported from Ganja.js to Lean 4.

Quaternion Mandelbrot

Visualizing the Mandelbrot set using Quaternion algebra Cl(0,2). Calculated in Lean and rendered to PNG.

Mandelbrot Boundary

Scatter plot of the Mandelbrot set boundary points. Rendered to SVG using LeanPlot.

3D Vector Field

Interactive 3D visualization of a vector field using Three.js (generated data).

Algebras Demonstrated

  • Cl(0,1): Complex numbers
  • Cl(0,0,1): Dual numbers (Automatic Differentiation)
  • Cl(0,2): Quaternions
  • Cl(1,0): Split-complex numbers
  • R3 / Cl(3,0,0): 3D Euclidean Geometric Algebra
  • PGA3D / Cl(3,0,1): 3D Projective Geometric Algebra
  • STA / Cl(1,3): Spacetime Algebra
  • CGA / Cl(4,1): Conformal Geometric Algebra
